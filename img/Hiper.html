Formato básico en HTML Este documento muestra las características básicas de formato de los documentos HTML. Explica cómo crear párrafos, colocar encabezamientos y utilizar el elemento STYLE. Párrafos en HTML Se crean utilizando la etiqueta P. El texto se distribuye entre los márgenes pasando a la siguiente línea cuando no cabe más en la actual. Se puede forzar un cambio de línea con el elemento BR, un elemento vacío que no tiene marca de cierre. Encabezamientos en HTML HTML contempla seis tipos predeterminados de encabezamientos, los correspondientes a los elementos H1, H2, ..., H6. Por defecto, disminuyen en ese orden en tamaño El elemento STYLE El elemento STYLE permite establecer las características de formato de los tipos de párrafos (elementos).
